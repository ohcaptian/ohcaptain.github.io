线性表的抽象数据结构
ADT list{ 

* 数据对象   
* 数据关系 (如<a1,a2...ai> i为位序)
* 数据操作  
    1 初始化操作
    2 插入操作
    3 删除操作

}

定义线性表

  • typedef struct {

    ElemType *elem;

    int length;

    }SqList;

    定义了一个无名结构体;并且给这个结构体取了一个别名Sqlist;之后就可以通过Sqlist来进行操作
    结构体中含有这个线性表的长度和指向元素的指针;

InitList
1 代码
Status initlist(sqlist &L)

{

L.elem = new ElemType[MAXSIZE];

if (!L.elem)

exit(OVERFLOW);

L.length=0;

return OK;

}
为线性表开创连续的存贮单元;
elem指向数组的第一个元素的地址;
L.length=0表示这个表长度为0;还有存放任何元素;
